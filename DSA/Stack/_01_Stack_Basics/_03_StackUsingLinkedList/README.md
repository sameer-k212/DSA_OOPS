# Stack Using LinkedList

## Implementation
Dynamic size stack using singly linked list.

## Operations
- **push(x)**: Add to top - O(1)
- **pop()**: Remove from top - O(1)
- **peek()**: View top - O(1)
- **isEmpty()**: Check empty - O(1)

## Advantages
- No size limit (dynamic)
- No overflow condition

## Complexity
All operations: O(1) time, O(n) space
