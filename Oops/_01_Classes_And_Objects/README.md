# Classes and Objects

## Concept
Classes are blueprints for creating objects. Objects are instances of classes.

## Key Topics
- Class definition and structure
- Object creation and initialization
- Instance variables and methods
- Scope of instance variables
- this keyword

## Files
- `Class$Object.java` - Basic class and object creation
- `Calculator.java` - Calculator class example
- `DemoClass.java` - Class demonstration
- `Student1.java` - Student class with properties
- `ScopeOfInstanceVariable.java` - Variable scope examples

## Example
```java
class Student {
    String name;
    int age;
    
    void display() {
        System.out.println(name + " - " + age);
    }
}
```
